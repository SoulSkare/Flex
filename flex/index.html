      Plugins in Google Chrome

      If you are experiencing issues setting up the Torque plugin, please note that Google Chrome has recently disabled the use of plugins in it's latest update.

      We're working hard on a fix but in the meantime, please use the following workaround to continue using webflix.me:

      1. Enter chrome://flags/#enable-npapi into the Chrome address bar

      2. Re-enable the NPAPI plugin setting by clicking 'Enable'

      3. Hit 'Relaunch Now' at the bottom of the page.

      Alternatively, you could make the switch to Safari or Firefox.

      For more information on Google's decision to disable this setting by default, read the blog post here.

      About webflix.me

      What is webflix.me?

      webflix.me is a free, web-based media player that can be used to watch HD movies and TV shows online at superfast speeds.

      How does it work?

      It works by leveraging peer-to-peer technology to stream video files from a network of other users - as opposed to a central server. This is known as the BitTorrent protocol and is a safe and widely used way of sharing files. More info »

      Is it legal?

      webflix.me is primarily a front-end for a torrent client, which in essence is legal. However, using this technology to share copyrighted materials is not legal in every country. For this reason, you must check that copyright law in your country permits you to use this service before you continue. For more information on the legality of file sharing in your country, click here.

      Which browsers are supported?

      webflix.me now supports all modern browsers, including Chrome, Firefox, Safari and IE10+ on Windows and Mac. Yay!

      Where does webflix.me get its data?

      There is no data stored on the webflix.me server other than the files required to run the application - all other data is pulled from other services. Torrents for movies are indexed by YIFY Torrents and torrents for TV shows are indexed by Kickass Torrents. This data is then combined with information from TMDB and OpenSubtitles.org to produce a clean and simple interface.

      Is this in any way affiliated with Popcorn Time or Time4Popcorn?

      No. This application was written completely from scratch to provide the best possible web-based torrent streaming service.

      What is Torque?

      Torque is a completely safe and legal plugin, developed by BitTorrent. It is an interface-less torrent client that runs in the background alongside your browser to provide torrent technology to webflix.me.

      Why am I being told that I need to install VLC media player?

      Certain TV shows are encoded in a way that means they can't be played natively by your browser. Because of this, any files that are in the .avi format will require the VLC player to be installed before they can be streamed on the web.
